# Butta Bomma Sarees & Collection 🥻

A full-featured Indian ethnic wear e-commerce website built with HTML, CSS & JavaScript.

## 🌐 Live Site
[https://saisanjaymanne.github.io/boutique/](https://saisanjaymanne.github.io/boutique/)

## 📁 Project Structure
```
boutique/
├── frontend/
│   ├── index.html        ← Main HTML file
│   ├── css/
│   │   └── style.css     ← All styles (Royal maroon & gold theme)
│   └── js/
│       └── main.js       ← All JavaScript logic
├── backend/
│   ├── server.js         ← Node.js / Express server
│   ├── package.json      ← Dependencies
│   └── .env.example      ← Environment variables template
└── README.md
```

## ✨ Features
- 🛍️ Shop page with 12 products across 5 categories
- 🔍 Filter by category, price range, occasion
- 📦 Product detail page with size & color selector
- 🛒 Shopping cart with GST calculation
- ❤️ Wishlist / Save items
- 👤 Login & Signup with localStorage
- 📞 Contact page with Google Maps (Ameerpet, Hyderabad)
- 📱 Fully responsive design

## 🗂️ Categories
- Sarees · Lehengas · Salwar Suits · Dupattas · Blouses

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend (template):** Node.js, Express.js
- **Deployment:** GitHub Pages

## 🚀 Run Locally
Just open `frontend/index.html` in your browser — no setup needed!
